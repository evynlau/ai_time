# Python 编程入门 - 6-12 岁儿童课程 🐍

## 🎯 课程介绍
这是一套为 6-12 岁儿童设计的 Python 编程入门课程，包含 21 天的学习内容。

## 📚 课程大纲

### 第一阶段：基础概念 (Day 01-05)
- Day 01: 认识编程 - 给机器人下命令
- Day 02: 让电脑说话 - print() 函数
- Day 03: 变量 - 小盒子收纳盒
- Day 04: 数据类型 - 文字、数字和颜色
- Day 05: 运算符 - 数学魔法棒

### 第二阶段：条件与循环 (Day 06-10)
- Day 06: 条件判断 - 如果...那么...
- Day 07: 布尔值 - 是或否
- Day 08: 比较运算符 - 谁大谁小
- Day 09: for 循环 - 重复做
- Day 10: 循环图案 - 用代码画画

### 第三阶段：列表和函数 (Day 11-15)
- Day 11: 列表 - 收藏盒
- Day 12: 列表操作 - 取和放
- Day 13: 函数 - 神奇咒语
- Day 14: 带参数的函数
- Day 15: 函数返回值

### 第四阶段：进阶内容 (Day 16-20)
- Day 16: while 循环
- Day 17: 嵌套循环
- Day 18: 字符串操作
- Day 19: 字典 - 标签盒子
- Day 20: 综合练习 - 学生管理系统

### 第五阶段：毕业项目 (Day 21)
- Day 21: 毕业项目 - 智能问答程序

## 🎮 使用方法
在每个 `dayXX` 目录下有 `lesson.py` 文件，可以直接运行学习。

## ️🎨 适合年龄
6-12 岁儿童，零基础学习编程

## 特色
- 大量趣味案例和练习
- 生动比喻帮助理解
- 循序渐进的学习路径
- 每个知识点都有实践

## 📖 开始学习
打开任意 `dayXX` 目录，运行 `lesson.py` 文件即可!
