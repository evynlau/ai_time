# 🐍 Python 编程入门课程 - 6-12 岁儿童

## 🎉 课程更新说明

这是一套**完整**的儿童编程入门课程，包含：

### ✅ 课程内容
- **21 天学习**：每天一节课程
- **配套练习题**：每节课后都有练习
- **答案参考**：完整的参考答案
- **教师/家长指南**：详细的教学指导
- **视频脚本**：配套教学内容

## 📚 课程大纲

### 📖 第一天到第五天：基础概念
- Day 01: 认识编程 - 给机器人下命令
- Day 02: 让电脑说话 - print() 函数
- Day 03: 变量 - 小盒子收纳盒
- Day 04: 数据类型 - 文字、数字和颜色
- Day 05: 运算符 - 数学魔法棒

### 🔄 第六天到第十天：条件与循环
- Day 06: 条件判断 - 如果...那么...
- Day 07: 布尔值 - 是或否
- Day 08: 比较运算符 - 谁大谁小
- Day 09: for 循环 - 重复做
- Day 10: 循环图案 - 用代码画画

### 🎭 第十一天到第十五天：列表和函数
- Day 11: 列表 - 收藏盒
- Day 12: 列表操作 - 取和放
- Day 13: 函数 - 神奇咒语
- Day 14: 带参数的函数 - 接收礼物
- Day 15: 函数返回值 - 得到结果

### 🚀 第十六天到第二十天：进阶内容
- Day 16: while 循环 - 不知道结束什么时候
- Day 17: 嵌套循环 - 循环套循环
- Day 18: 字符串操作 - 处理文字
- Day 19: 字典 - 标签盒子
- Day 20: 综合练习 - 学生管理系统

### 🎓 第二十一天：毕业项目
- Day 21: 毕业项目 - 智能问答程序

## 🎮 如何使用

### 给学习者:
1. 打开 `dayXX/lesson.py` 查看代码
2. 运行代码学习
3. 完成 `exercises/dayXX/exercise.py` 练习
4. 对比 `answers/README.md` 参考答案

### 给家长/教师:
1. 先阅读 `TEACHER_GUIDE.md`
2. 跟随孩子一起学习
3. 使用练习题巩固知识
4. 参考答案指导

## 🎯 学习目标

- ✅ 编程基本概念

- ✅ 列表、字典、函数
- ✅ 综合编程能力
