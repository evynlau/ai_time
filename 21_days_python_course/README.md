# 🐍 Python 完整学习路径 - 21 天从入门到实战

## 🚀 课程简介
这是一套完整的 Python 学习课程，包含 21 天的详细学习内容，从零开始到能够完成实际项目。

## 📊 课程结构

### 🔵 第一阶段：Python 基础 (Day 01-07)
**目标**: 掌握 Python 基本语法，建立编程思维

| 天数 | 主题 | 核心知识点 | 项目 |
|-----|------|---------|-----|
| Day 01 | Python 入门 | print(), 注释，环境 | 自我介绍程序 |
| Day 02 | 变量与数据类型 | str, int, float, bool | 个人信息卡片 |
| Day 03 | 运算符 | + - * / 比较 逻辑 | 简易计算器 |
| Day 04 | 输入输出 | input(), f-string | 互动问答程序 |
| Day 05 | 条件语句 | if elif else | 成绩判断系统 |
| Day 06 | for 循环 | range(), break/continue | 猜数字游戏 |
| Day 07 | 基础复习 | 综合练习、调试 | 计算器升级版 |

### 🟢 第二阶段：核心技能 (Day 08-14)
**目标**: 掌握 Python 数据结构、函数、文件操作

| 天数 | 主题 | 核心知识点 | 项目 |
|-----|------|---------|-----|
| Day 08 | 列表 | 索引、切片、方法 | 购物清单管理 |
| Day 09 | 字典 | 键值对、遍历 | 学生信息管理 |
| Day 10 | 函数基础 | 定义、参数、返回 | 数学工具函数 |
| Day 11 | 函数进阶 | 默认参数、*args | 通用工具函数 |
| Day 12 | 模块与包 | import、pip | 实用工具集合 |
| Day 13 | 文件操作 | 读写文件、路径 | 电子日记本 |
| Day 14 | 异常处理 | try-except、finally | 健壮的程序 |

### 🟡 第三阶段：进阶实战 (Day 15-21)
**目标**: 面向对象、数据处理、综合项目

| 天数 | 主题 | 核心技术 | 项目 |
|-----|------|---------|-----|
| Day 15 | 面向对象 1 | class, 对象，属性 | 学生管理系统 |
| Day 16 | 面向对象 2 | 继承、多态、封装 | 游戏角色系统 |
| Day 17 | 字符串处理 | 字符串方法、正则 | 文本处理工具 |
| Day 18 | 日期时间 | datetime、时间戳 | 生日倒计时 |
| Day 19 | 文件批量处理 | os 模块、shutil | 批量文件工具 |
| Day 20 | 简单 Web | flask 基础 | 个人网站 |
| Day 21 | 毕业设计 | 综合运用 | 个人展示项目 |

## 🎯 学习建议

### 学习时间分配
- **每天**: 2-3 小时
- **复习**: 每 3 天复习一次
- **项目**: 每周完成 1-2 个项目

### 学习方法
1. 先看教程了解概念
2. 运行示例代码
3. 完成练习题
4. 尝试修改代码
5. 完成项目实践

### 遇到问题
1. 先自己思考 15 分钟
2. 查阅官方文档
3. StackOverflow 搜索
4. 加入 Python 社区
