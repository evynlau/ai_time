## Day 04 - 数据类型
```python
name = "小明"
emoji = "🤖"
age = 8
price = 9.99

print(f"名字：{name} ({type(name).__name__})")
print(f"表情：{emoji} ({type(emoji).__name__})")
print(f"年龄：{age} ({type(age).__name__})")
print(f"价格：{price} ({type(price).__name__})")
```

## Day 05 - 数学运算
```python

print(f"10 - 3 = {remaining}")

print(f"4 × 4 = {squares}")

average = (85 + 90) / 2
print(f"平均：{average}")
```
